Difference between HTML and XHTML

Difference
HTML XHTML
HTML stands for hypertext markup language XHTML stands for extensible hypertext markup language
HTML is not case sensitive XHTML is case sensitive
HTML uses a format which is similar to document format XHTML uses markup language
All content can be included in the body element All content must be put in blocks
There is no hard rule on the structure of elements The structure of elements should be followed
Attribute values are not significant in HTML Attribute values are important in XTHML
All browsers support HTML There are few older versions which donot support XTHML
HTML has a lower future compatibility than XHTML XHTML has really good future compatibiity
HTML is less expensive XHTML is more expensive compared to HTML
